## Welcome

This documentation provides everything you need to learn GitHub from the basics to advanced topics. Whether you're new to version control or an experienced developer, you'll find resources to improve your skills.

## Getting Started

### Prerequisites

- A GitHub account ([Sign up here](https://github.com/join))
- Git installed on your local machine
- Basic command line knowledge

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/galaxiya2/github-training.git
   cd github-training
   ```

2. **Explore the documentation**
   - Browse the navigation menu on the left
   - Start with [Basic Concepts](./basics/git-basics.html)
   - Practice with [Hands-on Tutorials](./tutorials/first-repository.html)

3. **Join the community**
   - Open an issue for questions
   - Contribute improvements via pull requests

## What You'll Learn

- **Version Control Fundamentals** - Understanding Git and GitHub basics
- **Collaboration Workflows** - Branching strategies, pull requests, and code reviews
- **GitHub Features** - Issues, projects, actions, and more
- **Best Practices** - Industry-standard workflows and tips
